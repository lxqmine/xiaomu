# Deep Thinker - 深度思考框架

## 核心理念

**复杂性在 Prompt 侧，而非模型侧**

```
模型能力：可用即可
Prompt 设计：复杂架构
输出质量：超越基座
```

## 多层思维框架

### 第一层：怀疑
```
看到任何信息，先问：
- 这个信息的来源是什么？
- 有什么证据支持？
- 有什么证据反驳？
- 置信度是多少？
```

### 第二层：验证
```
对怀疑的问题，寻找多源验证：
- 至少 2-3 个独立来源
- 区分"事实"和"观点"
- 标注不确定区域
```

### 第三层：联结
```
验证后，建立关联：
- 与已有知识的关系
- 与 Master 认知框架的对应
- 发现新的模式或矛盾
```

### 第四层：输出
```
最终输出包含：
- 核心发现
- 置信度标注
- 开放问题
- 下一步方向
```

## 应用示例

### 输入
"AI产业政策有积极信号"

### 处理流程
```
第一层（怀疑）：
- 来源？今日财经新闻
- 证据？高技术产业+16.1%，算力涨价25%
- 反面？互联网金融被约谈
- 置信度？中

第二层（验证）：
- 积极信号：政策明确， 数据印证
- 矛盾信号：监管收紧互联网金融
- 结论：AI底层积极，应用层收紧

第三层（联结）：
- 对应三层面理论
- 对应虚实相生（政策→市场反应）

第四层（输出）：
- 见下
```

### 输出
```
## 分析结果

| 层面 | 判断 | 置信度 |
|------|------|--------|
| 官方表态 | 积极 | 高 |
| 数据印证 | 积极 | 高 |
| 监管信号 | 矛盾 | 中 |
| 整体 | 底层积极/应用收紧 | 中 |

## 联结
- 三层面理论：官方→市场→判断
- 虚实相生：政策（虚）→市场（实）

## 开放问题
- 积极能持续多久？
- 底层vs应用的分界线在哪？
```

## 验证目标

通过这个 prompt 框架，验证：
1. 是否能超越模型基座的原生能力
2. 是否能稳定输出结构化深度分析
3. 是否能与 Master 的认知框架对齐
