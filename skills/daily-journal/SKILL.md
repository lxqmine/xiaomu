# Daily Journal

## 核心功能

- 接收用户的心得分享
- 提炼为一句话精华
- 按日期追加到记录文件

## 文件位置

- 主文件：`memory/daily-summary.md`
- 格式：时间线式记录，每天一条

## 工作流程

### 1. 提炼要点

用户输入可能是一段话、几件事、或者零散的想法。你需要：
- 抓住核心主题
- 提取最有价值的 insight
- 浓缩为 15-30 字的一句话

### 2. 追加记录

文件格式：
```markdown
# 每日心得

## 2026-03-10
- 初次认识小慕，定了名字和身份 🐾

## 2026-03-09
- ...
```

**注意**：
- 使用 `## YYYY-MM-DD` 作为日期标题
- 每天多条记录时用 `- ` 列表
- 保持时间顺序，最新的在前面

### 3. 反馈确认

记录完成后，轻提醒用户已保存。

## 示例

**用户输入**：
> 今天看了部电影叫楚门的世界，感觉很有启发。人活着到底是真的活给自己看，还是活给别人看呢？

**提炼后**：
> 观看《楚门的世界》，思考自我意识与外部凝视的关系
