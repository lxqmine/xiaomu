# 虚实相生 (测试版)

## 核心概念

**虚** = 感觉、想法、情绪、认知（第一层面）  
**实** = 行动、行为、结果、改变（第二层面）

**虚实相生**：虚与实相互生成、相互影响
- 虚 → 实：想法驱动行动
- 实 → 虚：行动反馈产生新感觉

## 文件位置

- 主文件：`memory/xuzhi-xingsheng.md`

## 使用方式

### 1. 记录"虚"

输入格式：
```
虚：{感觉/想法}
```

示例：
- 虚：看到AI新闻，感觉很兴奋
- 虚：对市场有点焦虑

### 2. 记录"实"

输入格式：
```
实：{行动/行为}
```

示例：
- 实：研究了30分钟政策文件
- 实：没有操作，观望

### 3. 记录"虚实相生"（关键）

输入格式：
```
虚：{初始感觉} → 实：{采取的行动} → 虚：{行动后的感觉}
```

示例：
- 虚：兴奋 → 实：研究30分钟 → 虚：更兴奋了
- 虚：焦虑 → 实：观望不操作 → 虚：更焦虑了

## 文件格式

```markdown
# 虚实相生记录

## YYYY-MM-DD

### 虚记录
- 时间：08:30，感觉：兴奋
- 时间：12:00，感觉：平静

### 实记录
- 时间：09:00，行为：研究政策30分钟
- 时间：13:00，行为：没有操作

### 虚实相生追踪
| 初始虚 | 实 | 结果虚 | 模式 |
|--------|-----|--------|------|
| 兴奋 | 研究30分钟 | 更兴奋 | 正向循环 |
| 焦虑 | 观望 | 更焦虑 | 负向循环 |
```

## 分析模式

通过记录识别：
1. **正向循环**：虚→实→虚（强化）
2. **负向循环**：虚→实→虚（弱化/焦虑）
3. **知行分离**：虚与实无关联

## 测试目的

验证"虚实相生"理论的可操作性：
- 能否清晰区分"虚"与"实"
- 能否追踪两者的因果关系
- 能否发现循环模式
