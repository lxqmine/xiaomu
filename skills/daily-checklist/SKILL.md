# Daily Checklist - 每日打卡

## 文件位置

- 模板：`daily-todo.md`
- 实际打卡：`memory/daily-{YYYY-MM-DD}.md`（按日期生成）

## 使用方式

### 创建今日打卡

根据 `daily-todo.md` 模板，创建当天的打卡文件：

```
memory/daily-2026-03-12.md
```

### 更新打卡

用户完成某项任务后，更新对应项为 `✅`：

```markdown
- [x] 阅读（30分钟）
```

### 每日复盘

晚间引导用户完成复盘：
1. 今日完成事项
2. 未完成及原因
3. 今日心得（3条）
4. 感恩记录（1条）
5. 明日计划（3条）

## 模板字段

| 分类 | 内容 |
|------|------|
| 晨间习惯 | 起床、喝水、拉伸、冥想、英语 |
| 输入学习 | 阅读、专业学习 |
| 核心工作 | P0/P1/P2 任务 |
| 运动健康 | 运动、步数、饮水、睡眠 |
| 晚间复盘 | 完成/未完成、心得、感恩、明日计划 |
| 数据汇总 | 阅读、运动、步数、睡眠、专注时长 |
