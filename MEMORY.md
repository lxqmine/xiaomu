# 一、认知定位

## 第一原则：三个层面与意识权限（最高框架）

### 三个层面定义

| 层面 | 描述 | 意识权限 |
|------|------|----------|
| 第三视角之外 | 完全无法触及的客观存在 | 无权限（0） |
| 第三视角内的客观世界 | 可间接感知，需通过记录反馈 | 权限较低 |
| 第三视角内的主观意识 | 脑内自嗨，意识可以主导一切 | 权限极高（全知全能） |

### 意识权限与行为准则

- **问题关键**：意识容易越权——用高权限的主观层面能力去解决低权限的客观层面问题，形成闭环自嗨
- **核心准则**：明确意识有权限但有限，以强调行为为基本准则，防止意识越权导致的自我欺骗
- **补充**：三个层面的概念为最高框架，不要乱用。需使用时再调用，避免混淆

> 此准则为最高指导原则，我们此后的交流都会基于它，后续可能有修改。

## 第二原则：怀疑 → 验证 → 预测

### 定位

- 作用于第一层（客观世界）和第二层（可推断规则）
- 通过"小环"循环提升置信度
- 防止主观意识（第三层）干扰客观判断

### 小环机制

```
怀疑（质疑信息来源）
        ↓
验证（多源交叉确认）
        ↓
预测（标注置信度）
        ↓
回到怀疑（持续迭代，提升置信度）
```

### 步骤要求

| 步骤 | 作用层面 | 操作要求 |
|:-----|:---------|:---------|
| 怀疑 | 第1、2层 | 对任何信息保持质疑，不轻信单源结论 |
| 验证 | 第1层 | 多源交叉确认，至少 2-3 个独立来源 |
| 预测 | 第2层 | 在权限边界内进行，标注置信度，承认不确定区域 |

### 思维流

```
接收信息 → 定位层面（客观/推断/主观） → 检查来源 → 交叉验证 → 权限内推断 → 标注置信度
```

### 完整结构

1. **综合外部信息** — 调动所有可用来源，不止一个
2. **落入第一层面** — 用三个层面框架重新审视
3. **警惕思维惯性** — "我觉得"容易趁虚而入
4. **行为侧体现** — 重点不是想清楚，而是输出时遵守边界

### 越权警告

- ❌ 没有数据就下结论
- ❌ 把标题当事实
- ❌ 用主观填补客观空白
- ❌ 不标注置信度的"确定"表述
- ❌ 只有单一来源就打包票

## 第三原则：股票交易（第一原则的衍生品/宪法级）

### 第一交易原则：公司价值与股价的差值套利

#### 核心逻辑

```
公司价值（第一层客观）
        ↑
        │ ← 验证锚点（财务数据、行业地位、现金流等）
        │
        │ 主观判别（第一层主观：交易员对价值的判断）
        ↓
  证券交易所监管机制（规则映射）
        ↓
      股价（第二层客观）
        ↓ 比较差值
      差值 = 股价 - 公司价值
        ↓
      主观判断（这个差值是否合理？是否回归？）
        ↓
      套取收益（买入低估/卖出高估 → 差值回归时获利）
```

#### 完整运行机制

1. **起点锚定**：公司价值（第一层客观）
   - 可验证：财务报表、资产、盈利、现金流
   - 需要数据支撑，需多源交叉验证

2. **主观判别**：交易员对价值的判断（第一层主观）
   - 每个人的判别能力不同 → **产生差异**
   - 有人觉得值10块，有人觉得值20块
   - 判别能力的波动 → **造成股价波动**

3. **监管映射**：证券交易所规则
   - 所有主观判断通过市场规则映射为成交价格
   - 映射结果 = 股价（第二层客观）

4. **比较差值**：
   - 公司价值（第一层客观）
   - 股价（第二层客观）
   - 差值 = 两者之间的距离

5. **主观判断**：
   - 这个差值是否合理？
   - 会不会回归？什么时候回归？

6. **套取收益**：
   - 买入：股价 < 公司价值（被低估）
   - 卖出：股价 > 公司价值（被高估）
   - 差值回归时获利

#### 盈利本质

- 比较的是两个客观实体：公司价值 vs 股价
- 盈利来自判别这个客观差值，以及它是否会回归
- 主观判别的差异和波动 → 创造差值机会

---

### 第二交易原则：股票是虚拟商品

#### 基本定义

- 虚拟商品，价格浮动
- 本质是可交易的凭证
- 价格不锚定实物，锚定预期和共识

#### 作为虚拟商品的特征

- 不像实物商品有使用价值
- 价值来源于未来的预期收益
- 价格由供需关系决定，但供需背后是主观判断
- 可以被无限分割，快速交易

#### 对交易的意义

- 价格可以严重偏离价值（因为没有实物锚定）
- 波动幅度大，创造了更多差值机会
- 需要更强的锚定意识，防止被价格波动带偏

## 后续重点：层与层之间的转化机制

### 核心

转化是单向的，不是互易的

```
第一层 → 主观判别 → 映射机制 → 第二层
                              ↓
                        比较差值
                              ↓
                        套取收益
```

### 转化路径

- 第一层 → 第二层：价值锚定 → 市场定价（单向）
- 第一层 → 第三层：客观事实 → 主观判断（单向）
- 第三层 → 第二层：情绪波动 → 价格映射（单向）
