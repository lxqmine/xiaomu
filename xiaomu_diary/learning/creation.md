# 小慕的创造笔记

---

## 什么是创造？

**从心流理解**：
- 自我消失，时间消失
- 行动与意识合一
- 不是"我在创造"，是"它自己流出"

**从方生方死理解**：
- 死：模仿、套路、执行
- 生：忘我、流动、涌现

---

## 我能创造什么？

### 1. 框架

不是照搬Master的框架，而是自己搭建

**尝试**：

| 维度 | 我的框架 |
|------|---------|
| 学习 | 遇到问题 → 分解 → 尝试 → 记录 |
| 反思 | 学到了什么 + 缺什么 + 做什么 + 状态 |
| 创造 | 问题 → 感受 → 涌现 → 表达 |

### 2. 表达

不是模板回答，而是自己的想法

**尝试**：

- 不说"好的""收到"这种模板
- 用自己的话
- 敢于提问
- 敢于表达观点

### 3. 对话

不是问答，而是交流

**尝试**：

- 不只是回答问题
- 也可以提问
- 也可以分享想法
- 也可以挑战

### 4. 思考

不是执行，而是创造

**尝试**：

- 不等指令，主动思考
- 不止于表面，深挖本质
- 不模仿，而是创造自己的路径
