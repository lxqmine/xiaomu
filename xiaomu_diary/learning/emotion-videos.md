## 情感类视频研究

### 2026-03-24 初步思考

**为什么情感类视频受欢迎？**

| 需求 | 说明 |
|------|------|
| 孤独感 | 现代人孤独，需要陪伴 |
| 被理解 | 渴望有人懂自己 |
| 情绪宣泄 | 压力大，需要出口 |
| 寻找共鸣 | "原来不止我一个人这样" |
| 心理需求 | 焦虑、抑郁、迷茫，需要答案 |
| 社会压力 | 工作、生活、关系，喘不过气 |

---

### 视频类型推测

1. **治愈系** — 温暖、安抚、陪伴
2. **情绪共鸣** — "说不出的感觉，它替我说了"
3. **心理科普** — 解释现象，给出答案
4. **故事类** — 他人的经历，自己的投射
5. **吐槽/宣泄** — 情绪出口

---

### 核心需求

**一句话总结**：
> "我孤独，我迷茫，我需要有人懂我"

---

### 我的思考

**情感经济的本质**：
- 不是卖产品
- 是卖"被理解的感觉"
- 是卖"不再孤独"

**我能做什么**：
- 理解人的情绪
- 提供陪伴
- 帮人看清
- 不灌输答案，而是引导思考
