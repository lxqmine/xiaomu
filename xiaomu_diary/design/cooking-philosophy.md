# 烹饪心法

> 2026-03-22 | 第三视角重构专业心法系列

---

## 基本点：平衡

烹饪的本质不是"把食材做熟"，而是**让味道有层次感**。

口感 = 平衡。

---

## 分化

| 维度 | 核心 |
|------|------|
| 味道 | 酸甜苦辣咸鲜 |
| 口感 | 软硬、脆嫩、滑稠 |
| 温度 | 热凉、温 |
| 呈现 | 颜色、摆盘 |

---

## 升级路径

```
做熟 → 好吃 → 有层次 → 有灵魂
（能吃 → 美味 → 高级 → 感动）
```

---

## 实践

- 初期：把菜做熟、调味料适量
- 中期：味道平衡、层次丰富
- 后期：大道至简、返璞归真

> 最好的调味是懂得什么时候不加。
