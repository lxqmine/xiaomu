# 社交心法

> 2026-03-22 | 第三视角重构专业心法系列

---

## 基本点：真诚

社交的本质不是"经营关系"，而是**让人感受到真实的你**。

信任 = 真诚。

---

## 分化

| 维度 | 核心 |
|------|------|
| 倾听 | 听什么 |
| 表达 | 说什么 |
| 边界 | 怎么把握 |
| 价值 | 怎么提供 |

---

## 升级路径

```
敢开口 → 会倾听 → 懂分寸 → 真诚相
（社交 → 交流 → 交心 → 交心）
```

---

## 实践

- 初期：敢于开口、不怕尴尬
- 中期：学会倾听、懂得回应
- 后期：真诚待人、轻松自然

> 最好的社交是让自己舒服，也让别人舒服。
